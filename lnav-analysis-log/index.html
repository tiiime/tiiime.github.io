Log 分析神器

每次出了 bug，测试总是扔过来一个十几兆的 Log 日志过来分析，用 windows 的老哥们直接用 Notepad++ ，搜索过滤 tag 方便得很，效率很高灰常羡慕。苦逼的我拿着 sublime ，搜过各种插件还是被 Notepad++ 完爆。

终于！终于！终于！找到了个救星。lnav ，高亮过滤一应俱全 (还是命令行的，超酷

一般找到应用进程 filter-in，再把不相干 TAG filter-out，再之后慢慢搜索就好了，多个 TAG 可以正则过滤掉。需要多个应用一起分析就 filter-in 多个，再也不用被 Notepad++ 嘲笑了。

当然还有更多功能没用上，慢慢发掘吧。
